# Arduino Simulator

A full-stack monorepo for a browser-based Arduino electronics simulator.

## Structure
- `packages/frontend`: React 18, Vite, TypeScript, Tailwind CSS, Zustand, Konva, Monaco.
- `packages/backend`: Node.js, Express, TypeScript.

## Setup
1. Install dependencies: `pnpm install`
2. Start development servers: `pnpm run dev`
